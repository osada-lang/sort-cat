# Sort-cat (ねこそろえ)

猫を揃えるパズルゲーム「Sort-cat（ねこそろえ）」のプロジェクトリポジトリです。
WindowsからMacへの開発環境移行を完了し、Android版のストア申請を行いました。

## プロジェクトステータス

- **Android版**: 2026年4月5日 審査待ち (Pending Review)
- **パッケージ名**: `com.jirachi.nekozoroe`
- **開発フレームワーク**: Capacitor v8, HTML5/JavaScript
- **主な機能**:
  - AdMob 統合 (バナー広告)
  - Haptics (バイブレーション) フィードバック
  - 多言語対応 (日本語/英語)

## ディレクトリ構成

- `www/`: ゲーム本体のソースコード (HTML/JS/CSS)
- `android/`: Androidアプリのプロジェクトファイル
- `ios/`: iOSアプリのプロジェクトファイル (準備中)
- `store_assets/`: ストア掲載用の画像アセット（フィーチャーグラフィック、スクリーンショット）
- `privacy_policy_ja.md`: プライバシーポリシー（日本語版）
